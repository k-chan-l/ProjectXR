## 4-> 노드 자동생성을 위해 할 일

1. 기본적인 액터 생성 -> 캡쳐를 위해 방향, 카메라 포함 필요, 추가로 위치를 표현하기위해 원기둥형태의 메시 제작 필요
2. 언리얼엔진 내부에서 json을 바탕으로 정보를 읽는 코드 작성 -> Queue로 정리
3. World에 동적으로 해당 노드를 생성하는 코드 작성

## 5-> 캡쳐를 위해 할일(Unreal)

1. 캐릭터에서 사용할 함수 생성(자동촬영 시작, 캐릭터 이동, 캡쳐 및 저장)
2. 캡쳐할 위치 계산 함수 지정 -> x,y,z, 방향 -> 이동함수 내부에서 그냥 쓰면 될듯
3. 자동촬영시작 -> 노드 리스트를 순회 하며 다음 노드로 이동 -> 키 입력시 실행
4. 캐릭터 이동 함수 -> 현재 노드의 다음 노드를 순회하며 순차적으로 이동 하며 촬영
5. 캡쳐 함수 -> 현재 캐릭터의 시야를 바탕으로 이미지를 저장 현재노드\_다음노드.png형태로 이미지 저장
